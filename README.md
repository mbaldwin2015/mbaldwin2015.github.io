# mbaldwin2015.github.io

My personal portfolio website, built from scratch using HTML, CSS, and JavaScript.

## About

This is a static single-page portfolio site I designed and built by hand to showcase my projects and skills as a full stack developer. No site builders or templates were used — the layout, styling, and interactivity are all custom written.

## Built With

- **HTML5** — semantic page structure and content
- **CSS3** — custom styling with a dark/light section contrast layout
- **JavaScript** — sticky navbar behavior with scroll detection
- **Bootstrap 4** — responsive grid and component base
- **Shards UI** — lightweight Bootstrap theme for card and button styling
- **Devicon** — tech stack icons

## Features

- Responsive layout that works on mobile and desktop
- Smooth scroll navigation with a sticky navbar on scroll
- Project showcase cards with tech stack tags
- Skills section with language and framework icons
- Contact form via Formspree

## Projects Showcased

- **STFC Game Bot** — Python screen automation bot with image recognition and multi-threading
- **Arizona Cowgirl Candy Co.** — WordPress site designed and developed for a family-owned small business
